## How to run

Clone the repo, then run using npm.

```bash
git clone https://github.com/RomuloHe4rt/calculator-react.git && cd calculator-react
npm install
npm start
```

To run tests type

```bash
npm test
```

To run eslint checks

```bash
npm run lint
```

To deploy to github pages

```bash
npm run deploy
```
